## Overview

My game jam work is where I practice building quickly while keeping the final
project usable by the rest of the team. The playable builds on itch.io include
Last Oasis, A Totally Normal Bike Ride, and Alien Survivors.

## Practical Authoring Work

When a repeated task slows the project down, I build small workflow helpers.
Examples include a power line placement tool and a grid-to-actor placement tool
for Last Oasis.

## What The Work Shows

These projects are useful proof of iteration under pressure: scoping, debugging,
shipping playable builds, and recognizing when a small workflow improvement will
save more time than it costs.
